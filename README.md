# NBA Accreditation Assistant using IBM watsonx.ai and RAG

## Project Overview

This project is an AI-powered NBA Accreditation Assistant built using IBM watsonx.ai and Retrieval-Augmented Generation (RAG).

The assistant answers questions related to NBA accreditation using official NBA documents.

## Features

- Answers questions from NBA documents
- Uses Retrieval-Augmented Generation (RAG)
- Vector Index based document search
- IBM watsonx.ai foundation models
- Source-grounded responses

## Documents Used

- General Manual 2026
- NBA UG Engineering Tier I Manual
- Self Assessment Report (SAR)

## Architecture

User Query → IBM watsonx.ai Prompt Lab → Vector Index → NBA Documents → LLM → Response
